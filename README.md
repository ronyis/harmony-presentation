# The Quest for a Better Helpdesk

A technical presentation covering two core systems built at Harmony:

1. **The Agent Engine** — A modular AI agent framework built on PydanticAI with toolkits, gates, and lazy dependency injection
2. **The Communication Layer** — A unified threading system with SNS/SQS pub/sub connecting Slack, Teams, Portal, and Service Desk

## Running locally

```bash
npm install
npm start
```

Then open [http://localhost:8000](http://localhost:8000)

## Controls

| Key | Action |
|-----|--------|
| Arrow keys / Space | Navigate slides |
| `O` | Overview mode |
| `S` | Speaker notes |
| `F` | Fullscreen |
| `Esc` | Exit overview/fullscreen |
